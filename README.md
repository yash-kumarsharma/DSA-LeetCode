## 📖 How to Run a Solution

1. Navigate to the problem folder (e.g., `Arrays/`).  
2. Open the corresponding `.java` file (e.g., `TwoSum.java`).  
3. Run the `main()` method to test the solution with sample inputs.  

Example:
```bash
cd Arrays
javac TwoSum.java
java TwoSum
```
